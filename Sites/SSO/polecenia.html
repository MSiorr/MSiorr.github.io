6. Naprawić bootloader na dysku zawierającym partycję linuxową*
- psucie
sudo aptitude purge grub-pc

- naprawianie
Dodanie iso ubuntu i zbootowanie z niego z liveCD i
Znalezienie partycji z ubunty (ext4 zazwyczaj) - gparted - XY to numer partycji:
sudo mount -t ext4 /dev/sdXY /mnt
sudo mount --bind /dev /mnt/dev &&
sudo mount --bind /dev/pts /mnt/dev/pts &&
sudo mount --bind /proc /mnt/proc &&
sudo mount --bind /sys /mnt/sys
sudo chroot /mnt
grub-install /dev/sdX
grub-install --recheck /dev/sdX
update-grub
exit &&
sudo umount /mnt/sys &&
sudo umount /mnt/proc &&
sudo umount /mnt/dev/pts &&
sudo umount /mnt/dev &&
sudo umount /mnt

Lub:
sudo add-apt-repository ppa:yannubuntu/boot-repair
sudo apt-get update
sudo apt-get install boot-repair
boot-repair
